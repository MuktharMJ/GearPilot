# GearPilot

🌐 Live Demo: https://gearpilot-v1.vercel.app

GearPilot is a modern laptop recommendation platform built to help Indian students, developers, creators, and professionals find the right laptop without getting lost in confusing spec sheets.

The platform combines curated recommendations, advanced filtering, side-by-side comparisons, and favorites to simplify laptop buying decisions.

---

## Features

- Browse 50+ curated laptop models
- Smart filtering by budget and use case
- Side-by-side laptop comparison
- Favorites system with persistent storage
- Detailed laptop specification pages
- Search by laptop name, brand, and processor
- Responsive UI
- Pagination for improved browsing experience
- Modern animations and polished UX
